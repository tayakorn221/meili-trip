# 🏔️ เมยหลี่เสวี่ยซาน — แผนเที่ยวครอบครัว 7 วัน

> เว็บเพจแผนการเดินทางแบบไฟล์เดียว พร้อมแชร์ สำหรับทริป **เมยหลี่เสวี่ยซาน (梅里雪山 / Kawagebo)** มณฑลยูนนาน ประเทศจีน — ออกแบบสำหรับ **ครอบครัวที่มีเด็ก/ผู้สูงอายุ** งบประหยัด เน้นจุดชมวิวและการปรับตัวกับความสูง

## 🌐 เปิดดูเว็บ (Live Demo)

👉 **https://tayakorn221.github.io/meili-trip/**

## ✨ จุดเด่นของแผน

- 🗓️ **7 วัน 6 คืน** จังหวะไม่เร่งรีบ เดินไม่เกิน 4 ชม./วัน
- 🫁 **ปรับตัวกับความสูง 2 คืน** ที่แชงกรีล่า (3,200 ม.) ก่อนขึ้นเฟยไหลซื่อ — ปลอดภัยสำหรับผู้ที่เคยแพ้ความสูง
- 🌅 **ลุ้นอาทิตย์ขึ้นย้อมยอดคาวาเกโบ 2 เช้า** เพื่อเพิ่มโอกาสเห็นยอดเขา
- 💰 อยู่ในกรอบ **งบประหยัด** (~1,850–3,800 หยวน / ~9,250–19,000 บาทต่อคน)

## 🧭 เนื้อหาในหน้าเว็บ

| หัวข้อ | รายละเอียด |
|--------|-----------|
| ภาพรวม | สรุปทริปและไฮไลต์ |
| แผนเที่ยวรายวัน | ตารางวันต่อวัน + ชั่วโมงเดิน + ที่พัก |
| ที่พัก | การ์ดแต่ละคืน พร้อมระดับและราคา |
| อาหาร | ของกินเด็ดแยกตามเมือง |
| ค่าใช้จ่าย | ตารางสรุปต่อคน เน้นยอดรวม |
| ข้อควรระวัง & ทิปส์ | ความสูง / ฤดูกาล / ของที่ต้องเตรียม |

## 🛠️ เทคโนโลยี

- **HTML + CSS + Vanilla JS** ไฟล์เดียว ไม่มี build step ไม่พึ่ง framework
- **Google Fonts: Sarabun** (รองรับภาษาไทย) พร้อม fallback หากโหลดฟอนต์ไม่ได้
- **Responsive / mobile-first** — sticky nav, เมนูแฮมเบอร์เกอร์, ไฮไลต์เมนูตามการเลื่อน (scroll-spy), ตารางเลื่อนแนวนอนบนมือถือ
- โทนสี: น้ำเงินภูเขา `#1e3a5f` · ขาวอุ่น `#fafaf8` · ทอง `#c9a84c`
